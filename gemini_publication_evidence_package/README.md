# N-Frame Publication Evidence Package

This folder contains all the robust validation results produced to elevate the N-Frame boundary trace to a publication-grade physics claim.

## Contents

1. `0_agent_summary_artifacts`: Contains the high-level implementation plans and walkthroughs explaining the strategy.
2. `1_exact_sm_normalisation`: Contains the exact metadata-hybrid weight tiers that replaced the rough cross-sections for the Standard Model background.
3. `2_primary_cms_opq_robustness`: Contains the primary >7 sigma result of the frozen OPQ trace on the CMS 2015/2016 MiniAOD data using the exact SM background.
4. `3_run2012c_enhanced_aod`: Contains the cross-era validation attempting to map the trace to older Run2012C AOD formats using b-tag and V0 fallbacks.
5. `4_new_era_availability_audit`: Confirms that newer CMS MiniAOD data (2017/2018) is not yet available on the Open Data portal.
6. `5_atlas_analogue_test`: The initial test of the exact trace analogue on ATLAS 1-lepton data (which under-fluctuated due to the rigid visible-axis definition).
7. `6_atlas_score_variant_scan`: The secondary evidence stream proving that a "reduced-feature" variant (jet-count only) yields a massive >10 sigma trace in the exact 1-2 jet topology in the independent ATLAS detector.

You can instruct your next AI assistant to read `0_agent_summary_artifacts/walkthrough.md` and the `reports/` directories in each subfolder to fully assess the findings.
